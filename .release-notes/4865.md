## Add `--explain` flag to pony-lint

pony-lint now supports `--explain <rule-id>` to display a rule's description, whether it is enabled or disabled by default, and a link to the full documentation on the ponylang website.

```
$ pony-lint --explain style/dot-spacing

style/dot-spacing (enabled by default)

  no spaces around '.'; '.>' spaced as infix operator

  https://www.ponylang.io/use/linting/rule-reference/#styledot-spacing
```
